# My dotfiles

1. **.zshrc**
    - Configures Zsh shell.

2. **kitty**
    - Terminal emulator config.

3. **ohmyposh**
    - Prompt theme engine for shell.

## Requirements

### Git

```
pacman -S git
```
### Stow
```
pacman -S stow
```
## Installation
```
$ git clone git@github.com:saLucasRd/dotfiles.git
$ cd dotfiles
```

```
$ stow .
```
